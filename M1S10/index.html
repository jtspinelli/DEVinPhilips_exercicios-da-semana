Módulo 1 - Exercícios da Semana 10

  • Aprimorar o NotificationApp da M1S08

    Adicionar componente de filtro para as notificações.
    A componente deve apresentar 3 botões, para filtrar as notificações por: - Todos - Não lidos - Lidos

    Utilizar o decorator Output para criar um EventEmitter na componente de filtro.

    A lista de notificações deve ser movida para uma nova página, chamada Home.

    Devem ser criadas rotas para cada tipo de filtro (todas as variações abaixo devem enviar para HomeComponent):
    - home/todos
    - home/lidos
    - home/nao-lidos

    A componente de listagem das notificações deve captar qual filtro está ativo através da rota atual (utilizar ActivatedRoute).
